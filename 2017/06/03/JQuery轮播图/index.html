JQuery 轮播图

  • fadeIn fadeOut
  • 思路：
    • 设置结构 一个大盒子包含图片（嵌套在a标签中）、左右标签、还有数字按钮
    • 设置第一张图片显示其他的隐藏
    • 设置主要功能需求 定义变量i 和定时器
    • 鼠标进过数字按钮 clearInterval 清除定时器，停止自动轮播
    • 左右箭头点击事件轮播 点左判断i=0时 i=8 i– 点右判断if（i=7）{ i=-1}i++
    • 封装定时轮播函数 ，方便以上方法调用
    • 封装显示函数，方便以上函数调用 使用的是fadeIn 和 fadeOut 动画

$.fn.lun = function() {

​ var i = 0;

​ var timer;

​ $(‘.ig’).eq(0).show().siblings(‘.ig’).hide();

​ // 调用显示函数

​ showTime();

​ $(‘#igs’).on({

​ mouseover: function() {

​ clearInterval(timer);

​ },

​ mouseleave: function() {

​ showTime();

​ }

​ });

​ // 鼠标经过数字切换相对应的图片

​ $(‘.tab’).hover(function() {

​ i = $(this).index();

​ show();

​ clearInterval(timer);

​ }, function() {

​ show();

​ })

​ // 点击右箭头

​ $(‘.bt2’).on(‘click’, function() {

​ clearInterval(timer);

​ if (i == 7) {

​ i = -1;

​ }

​ i++;

​ show();

​ showTime();

​ });

​ // 点击左箭头

​ $(‘.bt1’).on(‘click’, function() {

​ clearInterval(timer);

​ if (i == 0) {

​ i = 8;

​ }

​ i–;

​ show();

​ showTime();

​ });

​ // 轮播函数

​ function showTime() {

​ timer = setInterval(function() {

​ show();

​ if (i == 7) {

​ i = -1;

​ }

​ i++;

​ }, 2000)

​ };

​ // 显示函数

​ function show() {

​ }
