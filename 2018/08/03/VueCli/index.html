Cli 脚手架搭建工具

  • Vue webpack cli

vue-cli

为什么要用vue-cli

实际开发中,为了提高开发效率, 会用到一些工具,如webpack

webpack可以让我们快速的创建项目目录,自动将代码模块化

当然,webpack功能不限于此,如果要在vue的项目中使用webpack这个工具,

就需要用到了vue-cli这个工具

什么是vue-cli

使用vue-cli

项目目录说明

  • node_modules 项目依赖包
  • src 项目核心文件(项目核心代码都放在这个文件夹下)
    • assets 静态资源(样式类文件,如css、less、sass以及外部的js文件)
    • App.vue 根组件
    • main.js 入口文件
  • .babelrc babel配置参数
  • .editorconfig 代码格式
  • .gitignore git忽略文件
  • index.html 主页
  • package-lock.json
  • package.json
  • README.md 项目说明
  • webpack.config.js webpack配置文件

注意:

一个vue页面通常由三部分组成:模板(template)、js(script)、样式(style)

我们关心的重点是src中的文件夹

单文件组件

说明:

  1. *.vue 文件，是一个自定义的文件类型，用类似HTML的语法描述一个Vue组件。
  2. 每个.vue文件包含三种类型的顶级语言块
